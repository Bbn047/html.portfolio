My Projects

Voice to vision Guidence

I developed an advanced assistive technology system that combines a wearable device with a mobile application to enhance mobility and independence for visually impaired individuals. The wearable device, equipped with a camera and speaker, connects to the mobile app to provide real-time assistance. Once connected, the camera captures live video, which is processed using Python, YOLOv5, and OpenCV to detect objects in the environment. Instant audio feedback helps users understand their surroundings. Additionally, EasyOCR enables text extraction, seamlessly converting it to speech using pyttsx3. The system also features a low-latency audio alert mechanism, ensuring real-time notifications that enhance user awareness compared to traditional methods.

Evacuation Vehicle

The wireless-controlled autonomous evacuation vehicle is designed for military rescue operations, providing safe and efficient evacuation of injured personnel. The vehicle features a secure compartment for safe transportation and an advanced robotic arm that assists with boarding and obstacle clearance. A high-accuracy targeting system detects injured individuals and identifies potential enemy threats, ensuring quick and accurate responses. Built with Arduino Nano and NRF24L01 modules, the vehicle supports real-time wireless communication for seamless operation. The system is programmed in C and Python, optimized for efficiency to reduce processing time. A joystick provides precise control over both the robotic arm and vehicle, ensuring reliable performance in critical battlefield scenarios. This system offers a comprehensive, automated solution for military rescue.

Hand writing Robot

Built with Arduino Nano, CNC Shield V4, and A4988 stepper motor drivers, the robot provides 1/16 microstep resolution, enhancing precision. Temperature Monitoring System with animation

Developed a real-time temperature monitoring system using the DHT11 sensor with OLED and LCD displays to visualize live temperature data. The system includes LED indicators to signal temperature thresholds, providing hot and cool alerts for better monitoring and safety. For display functionality, the Adafruit GFX and SSD1306 libraries were used to render custom graphics on the OLED, while the LiquidCrystal_I2C library enabled seamless interfacing with a 16x2 LCD for clear text output. The entire project was programmed in C++ using Arduino IDE, ensuring efficient sensor data processing, real-time updates, and smooth output control. This system is useful for climate control applications and smart home automation.
